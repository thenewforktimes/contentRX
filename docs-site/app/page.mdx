# ContentRX

A content-design review system for engineering and product teams. The
evaluator runs in the tools you already use — the Figma plugin, the
CLI, the GitHub Action, the MCP server, and an API — and reports
findings as plain-language guidance: an issue, a suggestion, a
severity, a confidence.

The model behind the evaluator is private. The accuracy of the
evaluator is public. See [/accuracy](/accuracy) for the headline κ
numbers (with 95% confidence intervals, never combined into a
composite score) and [/calibration](/calibration) for the weekly log
that tracks how those numbers move.
